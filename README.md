# Automation Project

This project is a Python automation script that helps with various tasks such as folder management, document sorting, log file parsing, and menu-driven application functionality.

- The `automation-project` directory contains the main script `main.py` responsible for the automation tasks.
- The `tests` directory contains unit tests in `test_automation_project.py` to ensure the functionality of the script.

## Getting Started

To use the automation script, follow these steps:

1. Clone or download the project to your local machine.
2. Make sure you have Python 3.x installed.
3. Install the required dependencies by running `pip install -r requirements.txt` in the project root directory.

## Running the Script

To run the automation script, navigate to the `automation-project` directory and execute the following command:

```
python main.py
```

## Running Tests

The project includes unit tests to verify the functionality of the automation script. To run the tests, navigate to the `tests` directory and execute the following command:

```
python -m unittest test_automation_project.py
```
